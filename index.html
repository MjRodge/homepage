Random Quote Machine

A project completed as part of the FreeCodeCamp front-end certificate curriculum. In order to complete the challenge, the following must be achieved:

I can click a button to show me a new random quote.

I can press a button to tweet out a quote.

Rather than maintaining my own array of quotes, I used jQuery's getJSON method to pull a quote from Quotes on Design's API each time the button is pressed and to ensure that no page reload was required between clicks. Posting to Twitter functionality has been added to meet project specification.

Local Weather Viewer

A project completed as part of the FreeCodeCamp front-end certificate curriculum. In order to complete the challenge, the following must be achieved:

I can see the weather in my current location.

I can see a different icon or background image (e.g. snowy mountain, hot desert) depending on the weather.

I can push a button to toggle between Fahrenheit and Celsius.

Using HTML5 geolocation I have taken the user's current location and connected to the OpenWeatherMap API to retrieve a forecast. The same coordinates have been used to set background image as a Google Map view of the local area. Depending on the forecast, the image and description will adjust themselves to a matching icon.
